Frequently Asked Questions

Does this web launcher require game downloads? +
No! The web launcher streams and runs the game code directly inside Chrome's WebAssembly sandbox. You don't need to download or install the actual game on your laptop.
How do I operate the virtual smartphone mod? +
Use the Up Arrow key to pull the phone up. Use Arrow Keys to navigate, Enter to dial/trigger cheats, and Backspace/Escape to close the phone.
Is this project free and safe? +
Yes! The project is 100% open-source, runs locally inside your browser tab without sending any personal data, and is licensed under the MIT license.
